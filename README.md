
# Next Movie

https://yunu-next-movie.vercel.app/

React + Next.js 실습용 개인 프로젝트입니다.

- 영화 목록 확인(인기순, 평가순, 상영 중)

- PC, 모바일 반응형 UI

## Run Locally

Clone the project

```bash
  git clone https://github.com/yunuchoiii/next-movie.git
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
